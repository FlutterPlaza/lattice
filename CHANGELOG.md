## 0.1.0

- Initial release of the Lattice post-quantum Signal protocol implementation.
- Core cryptographic primitives: KEM (ML-KEM/Kyber), SIG (ML-DSA/Dilithium), PRF, Ext.
- SC-AKE and SC-DAKE protocol implementation.
- Key distribution server with REST API (Shelf).
- Flutter/Dart client SDK.
- Platform integrations: Firebase Cloud Run, Serverpod, Supabase.
- Interactive deployment CLI (Docker, AWS, Azure, GCP, Firebase).
- Server health monitoring and alerting tool.
- Documentation website with protocol specification.
- CI/CD pipelines for testing, publishing, and deployment.
