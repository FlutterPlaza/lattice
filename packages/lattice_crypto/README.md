# lattice_crypto

Post-quantum cryptographic primitives for the Lattice Signal protocol. KEM (ML-KEM/Kyber), SIG (ML-DSA/Dilithium), PRF, and randomness extraction.

## Usage

```dart
import 'package:lattice_crypto/lattice_crypto.dart';
```

## Installation

```yaml
dependencies:
  lattice_crypto: ^0.1.0
```

Part of the [Lattice](https://github.com/flutterplaza/lattice) monorepo.

See the [documentation](https://flutterplaza.github.io/lattice/) for details.
