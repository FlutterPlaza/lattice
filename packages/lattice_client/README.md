# lattice_client

Flutter client SDK for the Lattice post-quantum Signal protocol.

## Usage

```dart
import 'package:lattice_client/lattice_client.dart';
```

This package is internal to the Lattice monorepo and is not published to pub.dev.

Part of the [Lattice](https://github.com/flutterplaza/lattice) monorepo.

See the [documentation](https://flutterplaza.github.io/lattice/) for details.
